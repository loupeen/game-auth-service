# Game Auth Service

JWT-Cedar authentication and authorization service for Loupeen RTS Platform.

## Quick Start

```bash
npm install
npm run build
npm test
```

## Deployment

Via GitHub Actions:
- **Test**: Auto-deploys on push to main
- **QA**: Manual workflow dispatch 
- **Production**: Manual workflow dispatch with approval
